# Plan Review Skill

Review proposals before implementation.

Check:

- whether the plan has enough code evidence
- whether it matches existing architecture and ownership boundaries
- lifecycle, memory, session, task, and trace implications
- hidden coupling and rollback risk
- whether a smaller first step would be safer

Output:

- Verdict
- Key Risks
- Missing Checks
- Suggested Next Step
- Evidence
